# Blunex Digital Studio - Animated Landing Page

A premium, modern, and highly interactive landing page built to practice advanced UI/UX layout concepts and smooth web animations. 

## 🚀 Live Demo
- **Live Link:** [View Live Site](https://blunex-animated-landing.vercel.app/)

## ✨ Features
- **Premium GSAP Animations:** Fully synchronized entrance timeline that smoothly animates text, imagery, and structural frames on load.
- **3D Parallax Mouse Effect:** Interactive mouse tracking where the foreground elements (Robot Head) and background elements (Floating Stones) move dynamically to create visual depth.
- **Modern Glassmorphism:** Sleek frosted-glass navigation bars and social link containers with backdrop blurs.
- **Custom Frame Geometry:** Implements an asymmetrical layout with smooth concave/convex borders using modern CSS.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3 (Modern Flexbox/Absolute Positioning)
- **Animation Engine:** JavaScript & [GSAP (GreenSock Animation Platform)](https://greensock.com/gsap/)
- **Typography:** Orbitron & Plus Jakarta Sans via Google Fonts
